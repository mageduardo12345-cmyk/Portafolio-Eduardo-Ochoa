# Portafolio Universitario de Series Audiovisuales

Sitio estatico listo para subirse a GitHub y desplegarse en Vercel.

## Estructura

- `index.html`: portafolio principal
- `vercel.json`: configuracion minima para servir el sitio estatico

## Flujo recomendado

1. Crear un repositorio nuevo en GitHub.
2. Subir el contenido de esta carpeta.
3. Importar el repositorio en Vercel.
4. Cada actualizacion futura se hace editando `index.html` y volviendo a publicar.

## Nota importante

El HTML actual usa algunas imagenes remotas temporales. Si alguna no carga en produccion, conviene reemplazarla por archivos locales dentro del repositorio.
